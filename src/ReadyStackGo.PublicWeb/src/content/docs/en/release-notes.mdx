## Version 0.6.0

*Released: December 2024*

### Multi-User Support & DDD Architecture

This version introduces a completely new architecture:

- **SQLite Database** for persistent storage
- **Domain-Driven Design** with Bounded Contexts
- **Role-based Access Control (RBAC)**

### New Features

- **Users & Roles**
  - Multiple users per organization
  - SystemAdmin, OrganizationAdmin, Deployer, Viewer roles
  - Permission-based authorization

- **GitHub Actions Integration**
  - CI/CD workflows
  - Automatic Docker image builds
  - SBOM generation for containers

### Changes

- Migration from JSON files to SQLite
- New DDD architecture with Clean Architecture
- Improved API structure with MediatR
