Welcome to **ReadyStackGo** - the modern container management platform for easy deployment of Docker Compose stacks.

## What is ReadyStackGo?

ReadyStackGo is a self-hosted platform that lets you deploy and manage Docker Compose stacks with just a few clicks. Whether you're managing a single server or multiple environments - ReadyStackGo makes it easy.

<CardGrid>
  <Card title="Self-Hosted" icon="server">
    Full control over your data. Runs on your own infrastructure without external dependencies.
  </Card>
  <Card title="One-Click Deployments" icon="rocket">
    Deploy complex container stacks with a single click using pre-built templates.
  </Card>
  <Card title="Multi-Environment" icon="setting">
    Manage development, staging, and production environments from a single interface.
  </Card>
  <Card title="Web UI" icon="laptop">
    Intuitive web interface for stack management, monitoring, and configuration.
  </Card>

## Quick Start

Start ReadyStackGo with a single command:

```bash
curl -fsSL https://readystackgo.pages.dev/install.sh | sudo bash
```

After installation, ReadyStackGo is available at `http://localhost:8080`.
