## Version 0.6.0

*Veröffentlicht: Dezember 2024*

### Multi-User Support & DDD-Architektur

Diese Version führt eine komplett neue Architektur ein:

- **SQLite-Datenbank** für persistente Speicherung
- **Domain-Driven Design** mit Bounded Contexts
- **Rollenbasierte Zugriffskontrolle (RBAC)**

### Neue Features

- **Benutzer & Rollen**
  - Mehrere Benutzer pro Organisation
  - SystemAdmin, OrganizationAdmin, Deployer, Viewer Rollen
  - Berechtigungsbasierte Autorisierung

- **GitHub Actions Integration**
  - CI/CD Workflows
  - Automatische Docker Image Builds
  - SBOM-Generierung für Container

### Änderungen

- Migration von JSON-Dateien zu SQLite
- Neue DDD-Architektur mit Clean Architecture
- Verbesserte API-Struktur mit MediatR
