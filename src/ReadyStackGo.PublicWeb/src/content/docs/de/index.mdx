Willkommen bei **ReadyStackGo** - der modernen Container-Management-Plattform für einfaches Deployment von Docker Compose Stacks.

## Was ist ReadyStackGo?

ReadyStackGo ist eine self-hosted Plattform, mit der du Docker Compose Stacks mit wenigen Klicks deployen und verwalten kannst. Egal ob du einen einzelnen Server oder mehrere Umgebungen verwaltest - ReadyStackGo macht es einfach.

## Schnellstart

Starte ReadyStackGo mit einem einzigen Befehl:

```bash
curl -fsSL https://readystackgo.pages.dev/install.sh | sudo bash
```

Nach der Installation ist ReadyStackGo unter `http://localhost:8080` erreichbar.
