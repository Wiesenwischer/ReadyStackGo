ReadyStackGo kann auf verschiedene Arten installiert werden. Wähle die Methode, die am besten zu deiner Umgebung passt.

## Voraussetzungen

Bevor du mit der Installation beginnst, stelle sicher, dass dein System die folgenden Anforderungen erfüllt:

| Anforderung | Details |
|-------------|---------|
| **Betriebssystem** | **Ubuntu 20.04+** (offiziell unterstützt) |
| **Architektur** | x86_64 (amd64) oder ARM64 |
| **RAM** | Mindestens 1 GB (2 GB empfohlen) |
| **Speicherplatz** | Mindestens 5 GB freier Speicher |
| **Netzwerk** | Internetzugang für Image-Downloads |
| **Berechtigungen** | Root-Zugriff oder sudo-Berechtigung |

:::caution[Inoffizielle Distributionen]
Debian, CentOS und Fedora funktionieren möglicherweise, sind aber **nicht offiziell unterstützt**. Bei Problemen mit diesen Distributionen können wir nur eingeschränkten Support bieten.
:::

## Nach der Installation

Nach erfolgreicher Installation ist ReadyStackGo erreichbar unter:

```
http://<server-ip>:8080
```

Beim ersten Zugriff wirst du automatisch zum **Setup-Wizard** weitergeleitet.
