# Manifests

Dieser Ordner enthält Release-Manifeste für ReadyStackGo-Deployments.

Manifeste werden normalerweise durch den CI/CD-Prozess generiert und hier abgelegt.

Beispiel:
- `manifest-v0.1.0.json`
- `manifest-v0.2.0.json`

Siehe [Manifest-Spezifikation](../docs/Configuration/Manifest-Specification.md) für Details.
